# About

GalileoTify is a modern Android music application designed to provide a smooth listening experience across a wide range of devices.

The project focuses on stability, compatibility, clean user experience, and community-driven development.

Starting from version **15.5.5**, GalileoTify supports all major Android architectures, allowing installation on nearly every Android device currently available.

---

# Features

- Music streaming
- Offline playback support
- AI-powered recommendations
- Modern Material Design interface
- No subscription required
- Universal APK distribution
- Optimized performance
- Multi-language support
- Dark & Light theme support
