# About

**GalileoTify** is a modern music streaming application designed to deliver a smooth, lightweight, and accessible listening experience across a wide range of Android devices.

The project combines modern Android technologies, universal architecture support, intelligent music discovery algorithms, and advanced AI-powered lyric translation to help users enjoy music without language barriers.

Starting from version **15.5.5**, GalileoTify officially supports all major Android architectures, allowing installation on almost every Android device currently available.

---

# Features

### Music Experience

- Music streaming
- Offline playback support
- Modern Material Design interface
- Dark & Light theme support
- No subscription required
- Optimized performance
- Multi-language interface support

### Discovery System

GalileoTify uses recommendation algorithms to help users discover songs, artists, playlists, and albums based on listening activity.

- Personalized recommendations
- Trending content discovery
- Artist exploration
- Album suggestions
- Playlist recommendations

### AI-Powered Lyric Translation

GalileoTify includes an advanced AI translation system specifically designed for translating song lyrics while preserving context, meaning, tone, and artistic expression.
